@************************* Simple Registration template. Takes a reverse route, the name field, the email field, and the password field from a form. If form validation fails, then the fields will have Twitter Bootstrap error highlighting. Clients should set the "error" key in flash scope to provide feedback on errors. **************************@ @(reverseRoute: play.api.mvc.Call, firstField: Field, lastField: Field, emailField: Field, passwordField: Field, confirmField: Field)

Register

@helper.form(reverseRoute, 'role -> "form") {
@{firstField.error.map { error => error.message }}
@{lastField.error.map { error => error.message }}
@{emailField.error.map { error => error.message }}
@{passwordField.error.map { error => error.message }}
@{confirmField.error.map { error => error.message }}
}
